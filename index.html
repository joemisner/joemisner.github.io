Misner Splunk Tool

Connects to a Splunk Enterprise or Universal Forwarder instance via REST API, retrieving the instance's configurations, input status, apps, and other useful information for troubleshooting and analysis. This tool's intention is to grant easy access into remote Splunk instances where shell or web access may not be available, but the splunkd management port (default 8089) is accessible.

License: GPLv3
Download: https://github.com/joemisner/misnersplunktool/releases
Latest Version: 2017.07.26

Misner Trap Tool

Graphically build and send SNMP notifications to a remote SNMP manager, including Traps and InformRequests. Allows saving of traps for future use, as well as building snmptrap arguments for sending from a remote system. Supports SNMP versions 1, 2c, and 3.

Download: https://github.com/joemisner/misnertraptool/releases
